https://www.theteams.kr/recruit 사이트의 채용공고를 scraping 한다.

1. 사용자로부터 입력값을 받는다.
   - 입력값은 쉼표로 구분한다
   - 기본값은 '데이터', '백엔드' 이다
2. 다음페이지가 없을 때까지 계속해서 스크래핑 한다.
